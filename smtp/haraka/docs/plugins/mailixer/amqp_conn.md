mailixer/amqp_conn
========

Attach RabbitMQ connection object to `server.notes`

Configuration
-------------

* `config/mailixer.ini` - contain configuration of all mailixer's plugins
  * `[rabbitmq]` - RabbitMQ connection config
    * `host` - RabbitMQ host
    * `port` - RabbitMQ port
    * `user` - RabbitMQ username
    * `password` - RabbitMQ password
